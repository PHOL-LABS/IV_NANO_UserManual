\chapter{Product overview}\label{ch:product-description}

IV-Indicators Nano modules are compact button gauges with a shared enclosure and interchangeable presets. The hardware is sold as a bundled three-pack or as dedicated indicators for a single measurement role.

\section{Available variants}
\begin{itemize}
    \item \textbf{IV-Indicators gauge three-pack.} A complete bundle that contains voltmeter, barometer, and thermometer indicators for drivers who want a matched set.
    \item \textbf{IV-Indicators Voltmeter.} Ships with presets that monitor vehicle system voltage using a bar or dot display.
    \item \textbf{IV-Indicators Barometer.} Targets manifold or fuel pressure monitoring with profiles tuned for the VAG 03C906051A sensor.
    \item \textbf{IV-Indicators Thermometer.} Factory-calibrated to the Ossca 01176 NTC temperature sensor used in Volkswagen classics.
    \item \textbf{IV-Indicators Lambda and Boost.} Specialist gauges for air--fuel ratio and turbo pressure monitoring when paired with the recommended sensors and controllers.
\end{itemize}

\section{Shared capabilities}
Each indicator can be reconfigured using the supplied programmer and IV-Conf software. Owners can adjust colour gradients, define up to four segment transitions, and select dot or bar rendering. Calibration routines allow the gauge to be matched to aftermarket sensors or tuned for custom voltage ranges. Once configured, settings are stored on the device so that it remembers the chosen profile even after power is removed.
