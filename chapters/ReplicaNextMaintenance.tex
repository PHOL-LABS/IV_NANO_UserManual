\chapter{Configuring with the IV-Conf Android beta}\label{ch:android-config}

An experimental IV-Conf Android application ships with the indicator set. The beta is functional for flashing presets and editing colour schemes, but PHOL-LABS recommends the web pack for mission-critical adjustments.

\section{Connecting the programmer}
\begin{enumerate}
    \item Insert the external programmer into the indicator before attaching any cables.
    \item Connect the programmer to the Android phone with a USB-C OTG cable. If the handset uses a different connector, place a USB hub or USB-A adapter between the phone and the programmer.
    \item Launch the IV-Conf beta, confirm the USB permission prompt, and tap \texttt{USB Connect}. Wait until the status reports \texttt{Connected}.
\end{enumerate}

\section{Loading presets and modes}
\begin{enumerate}
    \item Open the navigation drawer (three-bar icon) and select \texttt{Settings}.
    \item Choose a preset that matches the physical indicator --- Volts, Baro, Thermo, Lambda, or Boost.
    \item Tap \texttt{Set parameters} and allow the transmission to complete. The indicator must stay still while data is sent.
    \item Use the on-screen toggles to switch between dot and bar indication modes and adjust the backlight intensity slider.
\end{enumerate}

\section{Editing gradients}
\begin{enumerate}
    \item Select the gradient style (default: \texttt{Gradient}).
    \item Choose the number of segments (up to four) and assign colours and transition positions to each segment.
    \item Press \texttt{Set parameters} again so the updated gradient is written to the indicator.
\end{enumerate}

\section{Beta limitations}
The beta application is under active development. Expect interface changes and occasional stability issues; fall back to the web pack if a setting is not applied correctly. Report reproducible bugs directly to PHOL-LABS so firmware and app updates can be aligned.
