\chapter{Package contents}\label{ch:package}

Each IV-Indicators Nano kit includes:

\begin{itemize}
    \item The programmed indicator (single unit or three-pack).
    \item Corresponding socket and short harness for integration with the vehicle loom.
    \item External programmer with USB-C connector.
    \item Quick-start leaflet containing warranty contact information.
\end{itemize}

Inspect the shipment upon arrival. Report missing items or shipping damage to PHOL-LABS Kft before proceeding with installation.
