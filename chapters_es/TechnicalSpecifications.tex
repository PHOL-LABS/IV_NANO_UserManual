\section{Preajuste barom\'etrico}
El indicador barom\'etrico est\'a dise\~nado para el sensor de presi\'on \texttt{03C906051A} del Grupo Volkswagen.
\begin{itemize}
    \item Alimentaci\'on: 5~V regulados.
    \item Rango de medici\'on: 10~bar absolutos.
    \item Salida anal\'ogica nominal: 0.4--0.5~V en reposo.
    \item Rosca mec\'anica: M10\,\texttimes\,1.0.
    \item Conector: VAG \texttt{8K0973703}.
\end{itemize}

\section{Preajuste de term\'ometro}
Los preajustes de term\'ometro se env\'ian calibrados para el termistor NTC Ossca\,01176, habitual en los sistemas de refrigeraci\'on del Golf~II. Utilice la tabla de resistencias de la \autoref{app:temperature-table} al validar sondas alternativas. Para sensores NTC personalizados, actualice el coeficiente beta y la resistencia nominal mediante las herramientas IV-Conf.

\section{Preajuste de turbo}
El indicador de turbo requiere el sensor Dacia \texttt{223657266R} (\texttt{161B0004}/\texttt{8200225971}).
\begin{itemize}
    \item Conector de tres pines (use un Bosch sellado \texttt{1928403966} o equivalente).
    \item Voltaje de alimentaci\'on: +5~V.
    \item Ventana de funcionamiento: 20--250~kPa.
    \item Salida anal\'ogica: aproximadamente +1.8~V en el punto nominal de sobrealimentaci\'on.
\end{itemize}

\section{Integraci\'on lambda}
IV-Indicators Lambda lee sensores de banda estrecha directamente, pero est\'a pensado para funcionar junto con un controlador de banda ancha en motores ajustados. Los controladores recomendados incluyen SLC Free, DIY-EFI TinyWB y Sigma Lambda Controller Free~2. Cada dispositivo convierte una sonda Bosch LSU~4.9 en una se\~nal lineal de 0--5~V que el indicador puede mostrar.
