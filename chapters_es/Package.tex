\chapter{Contenido del paquete}\label{ch:package}

Cada env\'io de IV-Indicators Nano contiene los elementos esenciales para la instalaci\'on:
\begin{itemize}
    \item Indicador de bot\'on o juego de indicadores (seg\'un la variante solicitada).
    \item Z\'ocalo para la conexi\'on al mazo de cables del veh\'iculo.
    \item Herramienta externa de programaci\'on con conectividad USB.
\end{itemize}
