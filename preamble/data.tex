%% Abstract (recommended length around 80-200 words)
\newcommand*{\ManualAbstract}{%
    This user manual documents the \IVIndicatorsShort{} family of modular indicators designed for Golf 2, Jetta 2, Golf 1, Jetta 1, Polo, and Audi 80 B2 vehicles.
    It explains how the voltmeter, barometer, thermometer, lambda, and boost modules share the same programmable platform, summarises the supplied hardware, and lists the recommended sensors.
    The guide details the safety rules for powering the system, the connection workflow for each sensor type, and the configuration steps using the IV-Conf Android beta and the IV-Conf web pack.
    Step-by-step instructions cover preset management, calibration ranges, reflashing, and indicator customisation, preparing the structure for upcoming firmware updates.
    Final sections summarise labelling, packaging, and storage recommendations and provide the temperature appendix for quick diagnostics.
}
