\section{IV-Conf web pack reflashing}\label{app:reflashing}
Verwenden Sie das Webpack für Firmware-Updates. Führen Sie die folgenden Schritte durch, ohne die Stromversorgung zu unterbrechen, sobald die Übertragung gestartet wurde.
\begin{enumerate}
    \item Öffnen Sie \url{https://phol-labs.com/iv} und verbinden Sie sich mit dem Programmer wie in \autoref{ch:web-config} beschrieben.
    \item Stellen Sie sicher, dass die Baudrate auf 115200 gesetzt ist, und klicken Sie auf \texttt{Connect MODBUS}. Warten Sie auf eine grüne Statusmeldung.
    \item Drücken Sie \texttt{Stop Send Data}, um eventuelle automatische Preset-Uploads zu stoppen.
    \item Klicken Sie auf das Firmware-Symbol, wählen Sie die mit Ihrer Anzeige gelieferte Update-Datei aus und bestätigen Sie die Eingabeaufforderung.
    \item Lassen Sie die Anzeige unberührt, bis der Fortschrittsbalken abgeschlossen ist und das Gerät neu startet.
\end{enumerate}
