\chapter{Csomag tartalma}\label{ch:package}

Minden IV-Indicators Nano csomag tartalmazza a telepítéshez szükséges alapvető elemeket:
\begin{itemize}
    \item Gombműszer vagy műszerszett (a rendelt változattól függően).
    \item Foglalat a jármű kábelkorbácsához való csatlakoztatáshoz.
    \item Külső USB-s programozó eszköz.
\end{itemize}
