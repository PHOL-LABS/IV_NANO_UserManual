\chapter{Package contents}\label{ch:package}

Chaque livraison IV-Indicators Nano contient l'essentiel pour l'installation :
\begin{itemize}
    \item Afficheur bouton ou ensemble d'afficheurs (selon la variante commandée).
    \item Douille pour la connexion au faisceau du véhicule.
    \item Outil de programmation externe avec connectivité USB.
\end{itemize}
