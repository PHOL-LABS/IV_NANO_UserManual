\chapter{Комплектация}\label{ch:package}

Каждая поставка IV-Indicators Nano включает всё необходимое для установки:
\begin{itemize}
    \item Кнопочный индикатор или набор индикаторов (в зависимости от заказа).
    \item Разъём для подключения к проводке автомобиля.
    \item Внешний программатор с USB-подключением.
\end{itemize}
